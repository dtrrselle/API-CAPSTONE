# 🌎 Holiday Explorer

Holiday Explorer is an API-powered web application that allows users to discover and explore public holidays from different countries using the Holiday API. The system enables users to search holidays by country and year, view upcoming holidays, and access real-time holiday information through API key authentication. This project was developed to demonstrate API integration, data retrieval, responsive web design, error handling, and collaborative development using Git and GitHub. The application is built using HTML, CSS, and JavaScript, providing a simple and user-friendly interface for exploring holiday information around the world.

## ✨ Features

- Search holidays by country and year
- View upcoming holidays
- Real-time data retrieval using Holiday API
- API key authentication
- Responsive and user-friendly interface
- Error handling for failed API requests
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS
- JavaScript 
- Holiday API
- Git & GitHub
